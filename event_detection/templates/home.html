Welcome to SocioPedia System.




Purposes:

We introduce SocioPedia, which is a real-time knowledge graph completion system using up-to-date extracted knowledge from social networks. SocioPedia aims to perform a real-time automatic light-weight data collection, extract reliable and highly interconnected up-to-date knowledge. Furthermore, SocioPedia provides a user-friendly and intuitive environment, which is optimized for user classes of knowledge graph contributors and domain experts/data analysts.
